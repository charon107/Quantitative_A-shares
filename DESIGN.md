# 设计系统 — A股股价数据可视化看板（深色终端风）

> 视觉方向：**深色 OLED 终端风**，贴合 Bloomberg / TradingView 的专业盘面工具。
> 唯一颜色来源为 [ui_theme.py](src/visualization/ui_theme.py) 的 `COLORS`，并与 [.streamlit/config.toml](.streamlit/config.toml) 同步。

## 颜色系统

### 背景与面板（深色）
- **背景（近黑）**：`#020617` (RGB: 2, 6, 23)        [OLED 主背景]
- **面板/卡片**：`#0F172A` (RGB: 15, 23, 42)         [KPI 卡、侧边栏]
- **次级面板**：`#1E293B` (RGB: 30, 41, 59)          [图表网格线]
- **边框**：`#334155` (RGB: 51, 69, 85)              [卡片/分隔线]
- **边框 hover**：`#475569` (RGB: 71, 85, 105)

### 文本
- **主文本**：`#F8FAFC` (RGB: 248, 250, 252)         [深色上 ≥ 7:1 对比]
- **次文本/中性**：`#94A3B8` (RGB: 148, 163, 184)    [标签、平盘]

### 涨跌主色（金融向）
- **上涨绿**：`#22C55E` (RGB: 34, 197, 94)   [深色上 WCAG AAA；伴随 ▲ 符号]
- **下跌红**：`#EF4444` (RGB: 239, 68, 68)   [深色上 WCAG AA+；伴随 ▼ 符号]
- **主色（=上涨绿）**：`#22C55E`             [强调/选中态]

### 辅助色
- **涨停黄**：`#FBBF24` (RGB: 251, 191, 36)  [涨停家数曲线/警示]
- **跌停蓝**：`#3B82F6` (RGB: 59, 130, 246)  [跌停家数曲线/波动率]
- **成交量副图**：`#1E3A5F` (RGB: 30, 58, 95) [深蓝，不抢眼]

## 排版

### 字体堆栈
- **正文/表格**：`"Fira Sans", "Segoe UI", -apple-system, BlinkMacSystemFont, sans-serif`
- **数字（K线、成交额、KPI）**：`"Fira Code", "IBM Plex Mono", monospace` [等宽，便于对齐数字]
- **标题**：同正文，但通过字重区分
- 字体经 CSS `@import` Google Fonts 加载（见 `ui_theme.inject_global_css`）。

### 字体大小 & 权重

| 用途 | 大小 | 权重 | 示例 |
|------|------|------|------|
| H1（页面标题） | 28px | 700 Bold | "大盘概览" |
| H2（tab 标题） | 24px | 600 SemiBold | "个股查询" |
| 卡片数字（市场宽度） | 36px | 700 Bold | "2500" (涨家数) |
| 卡片标签 | 14px | 500 Medium | "上涨家数" |
| 表格表头 | 14px | 600 SemiBold | "代码 \| 名称 \| 涨幅" |
| 表格正文 | 13px | 400 Regular | 数据行 |
| 时间戳/元数据 | 12px | 400 Regular | "更新于 2 小时前" |

### 行高
- **正文**：1.6 (充足的呼吸空间)
- **表格**：1.4 (紧凑但可读)
- **卡片内容**：1.5

## 间距系统

基于 4px 网格（Streamlit 默认为 16px）：

| 名称 | 值 | 用途 |
|------|-----|------|
| **xs** | 4px | 行内间距、图标旁边 |
| **sm** | 8px | 小元素间隙 |
| **md** | 16px | 卡片内部 padding、图表底部 |
| **lg** | 24px | 区块之间（大盘概览内的卡片组之间） |
| **xl** | 32px | Tab 之间、主要区域分隔 |

## 组件规范

### KPI 卡片（市场宽度 / 个股指标）— `ui_theme.kpi_card`
```
┌─────────────────────┐
│  上涨家数             │ ← 标签 (13px Medium, #94A3B8)
│  ▲ 2,500            │ ← 数字 (30px Bold mono, #22C55E for 涨)
│  +0.5%              │ ← 变化 (13px mono, 语义色 + ▲/▼)
└─────────────────────┘
背景：#0F172A
边框：1px #334155（hover 提亮至 #475569，translateY -1px）
圆角：10px；数字用 Fira Code 等宽
```

### K线图 — `dashboard.build_kline_fig` + `apply_chart_theme`
- **蜡烛**：阳线（涨）空心 #22C55E 描边、阴线（跌）实心 #EF4444；空心/实心形状冗余助色盲辨识
- **均线**：MA5/10/20/60 叠加，色板见 `ui_theme.MA_COLORS`（避开红绿），图例可点击开关；在完整序列上计算后再裁剪窗口
- **去空档**：`rangebreaks` 隐藏周末与节假日，蜡烛连续无大段空白
- **读价**：x+y 双向 spike 十字光标 + `hovermode="x unified"`（看盘式 OHLC/均线读数）；价格轴右置（金融惯例）
- **最新价**：虚线参考线 + 右侧涨跌色价签
- **成交量副图**：高度 26%，换算为「万股」、半透明降权，叠加量MA5
- **背景**：透明（贴合 #020617），**网格线**：#1E293B（深色，不喧宾夺主）

### 排行表格 — `ui_theme.style_ranking`
- **涨跌幅**：红绿文字 + 以 0 为中心的单元格条形（pandas Styler `.bar`）
- **数值列**：Fira Code 等宽；最新价 2 位、成交额→「亿」、换手率→「%」
- **行选择**：`st.dataframe(on_select="rerun", selection_mode="single-row")`，点击行 → 下方内联 K线速览
- **小数精度**：涨幅 2 位，成交额 2 位（单位 亿）

## 响应式设计

### 断点
- **桌面** (≥1200px)：Streamlit 默认布局（顶部 tabs）
- **平板** (768px-1199px)：表格变为 2 列或横向滚动（Plotly 自适应）
- **手机** (<768px)：
  - 大数字卡片堆叠为 1 列
  - 排行表格隐藏低优先级列（只显示 代码、名称、涨幅）
  - K线图高度减少 40%
  - 时间戳 font-size 减小到 11px

## 无障碍（A11y）规范

### 颜色对比（深色背景）
- **所有文本**：WCAG AA 标准（4.5:1 for 正文，3:1 for 大号文本）
- **涨/跌**：不仅依赖颜色，还加上 ▲/▼ 符号
- **主文本**：#F8FAFC on #020617，对比度 ≥ 7:1（AAA）
- **次文本**：#94A3B8 on #0F172A，对比度 ≥ 4.5:1

### 键盘导航
- **Tab 键**：可以在所有交互元素间导航（搜索框 → 排行表格 → 时间戳）
- **Enter 键**：在搜索框执行搜索
- **箭头键**：排行表格可排序（↑/↓ 按列排序）

### 屏幕阅读器标签
- 大数字卡片：`aria-label="上涨家数：2500，变化 +0.5%"`
- K线图：`aria-label="上证指数 K 线图，2025 年 1 月至今"`
- 排行表格：`<table role="grid">` + `<thead>` 清晰标题
- 搜索框：`<label htmlFor="search-input">股票代码或名称</label>`

### 触摸优化
- **按钮/可点击区域**：最小 44×44px
- **表格行**：最小 40px 高度
- **链接/排序头**：padding 增加使目标更大

## 动画 & 过渡

### 加载动画
- **骨架屏**：灰色占位符，呼吸式脉冲（opacity 0.5 → 1，周期 1.5s）
- **旋转图标**：SVG 旋转，360° per 0.8s（线性）

### 过渡与减少动效
- **交互过渡**：卡片/标签/按钮 hover 用 150–300ms `ease`（边框提亮、轻微位移）
- **加载反馈**：慢加载（全市场 parquet 扫描）用 `st.spinner`
- **减少动效**：`@media (prefers-reduced-motion: reduce)` 下关闭所有过渡

## 品牌基调

看板应该感觉**专业、清晰、信息密集**——像终端交易员的工具，不是消费级应用。

- **没有**：emoji 作图标、彩虹色、浮夸阴影、装饰性元素
- **有**：近黑背景、有限颜色（仅涨绿/跌红/中性）、有力的等宽数字排版、表格为王
- **强调色克制**：仅上涨绿 `#22C55E` 作主色（选中态、tab 高亮、关键 KPI）

参考：TradingView、彭博终端（功能性 ≫ 视觉精美）。
