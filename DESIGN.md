# 设计系统 — A股股价看板（Anthropic 浅色暖调）

> 视觉方向：**米白暖调 + 陶土强调 + 衬线/无衬线混排**，贴合 Anthropic 官网的克制、编辑感、高级气质。
> 颜色与字体真源：[frontend/tailwind.config.ts](frontend/tailwind.config.ts) 与 [frontend/src/theme/echarts.ts](frontend/src/theme/echarts.ts)（ECharts 同步）。

## 颜色系统

### 背景与面板
- **页面背景（奶油）**：`#FAF9F5`
- **卡片**：`#FFFFFF`
- **次级面板 / 浅米**：`#F0EEE6`
- **暖灰边框**：`#E5E1D8`
- **图表网格线**：`#ECE8DF`（虚线）

### 文本
- **主文字（墨色）**：`#1A1A18`
- **次文字**：`#6B6760`

### 强调与语义
- **强调（book cloth 陶土）**：`#CC785C`（hover `#BD5D3A`）— 选中态、KPI 强调、链接、图表主线
- **上涨**：`#C84B31`（陶红）— A股惯例「红涨绿跌」
- **下跌**：`#3A8C5F`（沉稳绿）
- **辅助**：琥珀 `#D9A441`（未结束样本）、灰蓝 `#5B7C99`（波动率）

> 涨跌采用 A股惯例「红涨 / 绿跌」，配合 KPI 文字色区分。

## 排版

- **标题（衬线）**：`"Source Serif 4", "Noto Serif SC", Georgia, serif`
- **正文（无衬线）**：`"Inter", "Noto Sans SC", system-ui, sans-serif`
- **数字 / 代码（等宽）**：`"IBM Plex Mono", monospace`，开启 `tnum`（表格数字对齐）
- 中文必须挂 **Noto Serif SC / Noto Sans SC**（Anthropic 原版 Tiempos/Styrene 无中文字形），经 Google Fonts 加载（见 [index.html](frontend/index.html)）。

| 用途 | 字号 / 字重 | 字体 |
|------|------------|------|
| H1 页面标题 | 20px / 600 | 衬线 |
| H2 区块标题 | 18-20px / 600 | 衬线 |
| KPI 数字 | 30px / 600 | 等宽 |
| 卡片标签 | 13px / 500 | 无衬线 |
| 表格正文 | 13-14px / 400 | 无衬线 + 数字等宽 |

## 质感

- **卡片**：白底、`rounded-2xl`、1px 暖灰边框、柔和阴影 `0 1px 3px rgba(26,26,24,.06)`；hover 抬升为 `lift` 阴影。
- **留白**：区块间 24px，卡片内 20px；整体宽留白、编辑感。
- **过渡**：hover 150-300ms；`prefers-reduced-motion` 下关闭。
- **无装饰**：不用 emoji 图标、不用彩虹色、不用浮夸阴影。

## 组件

- **KpiCard**：标签（次文字）+ 大号等宽数字（语义色）+ 可选 delta。
- **导航**：顶部 tab，激活态陶土文字 + 下方陶土短下划线。
- **SearchBox**：圆角输入，focus 陶土描边 + 陶土光环；防抖 250ms 下拉。
- **表格**：暖灰分隔线，行 hover `panel2`，选中行陶土淡底；涨跌幅语义色。
- **图表（ECharts）**：透明背景贴合奶油底、暖灰虚线网格、墨字、陶土主线；K线阳线（涨）空心陶红描边 / 阴线（跌）实心沉稳绿，MA5/10/20/60 中性色板（避开红绿）；x 轴 category 天然跳过周末空档。

## 品牌基调

专业、克制、可信，像研究工具而非消费应用。强调色克制（仅陶土作主强调），信息密度适中，呼吸感充足。
